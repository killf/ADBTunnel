# ADBTunnel — 产品概述

## 一、项目背景

在移动开发、自动化测试、远程运维等场景中，工程师经常需要对 Android 设备执行 ADB 命令。然而，传统 ADB 连接要求调试机与设备处于同一局域网，或必须物理使用 USB 线缆，在以下场景中面临严重障碍：

- 设备在远程 IDC 机房或用户手中，无法直接触达；
- 自动化测试平台需要通过 HTTP API 驱动设备，而不依赖 ADB 二进制；
- 团队共享设备池，不同成员需从各自网络环境发起控制；
- CI/CD 流水线在云端运行，需操控物理机房中的真机。

ADBTunnel 正是为解决上述问题而设计：在 Android 设备上安装一个轻量 App，通过 WebSocket 隧道将设备能力注册到云端服务器，服务器再以**标准 HTTP API** 对外暴露，实现"随时随地通过 HTTP 操控任意已注册的 Android 设备"。

---

## 二、核心价值

| 价值点 | 说明 |
|--------|------|
| 无需公网 IP | 设备主动出站连接，穿越 NAT/防火墙 |
| 标准 HTTP 接口 | 任何语言/工具均可调用，无需安装 ADB |
| 多设备并发 | 一台服务器可同时管理数百台设备 |
| 低侵入 | 设备无需 Root，仅需开启 ADB 调试 |
| 实时双向 | WebSocket 隧道支持流式输出与交互 |
| 安全可控 | Token 认证 + TLS 加密 + 设备白名单 |

---

## 三、目标用户

1. **移动测试工程师**：通过 HTTP API 在 CI 中操控真机，替代复杂的 ADB 环境配置；
2. **移动运维团队**：远程排查现场设备问题，无需派人到场；
3. **设备农场运营者**：统一管理大批 Android 设备，提供 HTTP 接入层；
4. **个人开发者**：在任意网络环境操控家中或办公室的 Android 设备。

## 五、功能范围（MVP）

### 5.1 Android App
- 检测设备 ADB 调试状态（USB 调试 / 无线调试）
- 引导用户开启开发者模式与 ADB 调试
- 前台 Service 维持 WebSocket 长连接（含断线重连）
- 设备注册：上报设备 ID、型号、Android 版本等信息
- 接收服务端 WebSocket 帧，执行 ADB 命令并返回结果
- 支持的命令类型：
  - `shell` — 执行 Shell 命令
  - `screencap` — 截图
  - `screenrecord` — 录屏（流式）
  - `install` / `uninstall` — APK 管理
  - `push` / `pull` — 文件传输
  - `logcat` — 日志流式输出
  - `getprop` — 属性查询

### 5.2 Server
- 设备注册 / 注销管理
- HTTP API（见第四章协议设计）
- WebSocket 会话多路复用（单连接多并发命令）
- 请求超时与会话清理
- 设备心跳检测（Ping/Pong）
- Token 鉴权
- 结构化日志 & 指标暴露（Prometheus）

## 六、技术选型理由

### Android — Java
- 成熟的 Android 生态，文档丰富；
- `OkHttp` 提供稳定的 WebSocket 客户端；
- `Android Foreground Service` + `WorkManager` 保活机制完善；
- 兼容 Android 7.0+（API 24+）。

### Server — Rust
- 异步高性能：`Tokio` 事件循环，单机支撑数千并发 WebSocket；
- 内存安全：避免 C/C++ 的内存问题，减少运维风险；
- `Axum`：基于 `Tower` 的轻量 HTTP 框架，与 `Tokio` 深度集成；
- `tokio-tungstenite`：原生异步 WebSocket 库；
- 编译产物为单一静态二进制，部署极简。

### 隧道协议 — WebSocket over TLS
- 基于 HTTP/1.1 升级，穿越绝大多数防火墙和代理；
- 全双工，支持服务端主动推送命令；
- 帧格式标准，方便调试与扩展；
- 相比原始 TCP 隧道，更易过 CDN / 负载均衡。

---

## 七、非目标（Out of Scope）

- 不支持 iOS 设备；
- 不提供完整的 ADB over TCP 协议代理（即不支持 `adb connect` 标准二进制，MVP 仅 HTTP API）；
- 不负责 Android 设备的 Root 权限获取；
- 不提供 Web UI 控制台（路线图中规划）。
